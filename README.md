# Library Management App

## Overview

Simple library management application that allows users to add, view, and manage books. 
Users can input details such as the book title, author, number of pages, and whether they have read the book.

![Screenshot of Library App](./assets/Screenshot.png) 

## Features

- **Add New Books**: Users can input details for new books through a dialog form.
- **Display Books**: All added books are displayed showing their title, author, number of pages, and read status.
- **Toggle Read Status**: Users can mark books as read or not read.
- **Remove Books**: Users can remove books from the library.

## Technologies Used

- HTML
- CSS
- JavaScript
